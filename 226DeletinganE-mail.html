2.2.6. Deleting an E-mail

Users can delete mails in inbox, sent mailbox, and other private mailboxes.


    1. Click the [E-Mail] button in main menu bar.
    2. Click the [Inbox] menu in the left toolbar.
    3. Click an email.
    4. Click [Delete] button, and then.
    5. A [Are you sure you want to move the folder?] popup window is displayed.
    6. Click [YES] button to confirm.





[Note] There are options for "Deleted Mail Management"


Restoring Deleted Mail

    1. Go to ‘Trash’ folder and select the mails to be restored.
    2. Click the [Archive] button, and a [Choose Folder] window will pops up.
    3. Select the target folder where the restored mail will be saved and click the [Apply]button, and [OK]button when a popup window says “Are you sure you want  move the folder?”
